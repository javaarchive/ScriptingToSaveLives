FAQ

Scripting To Save Lives Frequently Asked Questions

1. How Does It Work?

When you submit code, it is reviewed and if it is deemed good for the website, we will add it. Your code will add to the clean look of the website and make donors feel more confortable when making a donation.

2. How can I contribute?

You can contribute by opening a Pull Request to our Github Repo or upload some code on our Developer Panel.

3. How do donations work?

You can donate to noe of the maney charities we support by clicking here, your donation will go directly to the charity.

4. I don't have money, how can I help?

You can help program this website by clicking here, if you don't have coding skills, no worries you can donate your CPU to help find a Corona Virus Cure.

5. Why do you not process the payments ourselves.

Why do we need to? Even if we did we'd lose money from the fees and we also don't want to seem like a quick money grab so we're linking directly to the charities.

6. Is the website open-source?

Why wouldn't it be. Our website is open-source and can be found here. Every developement version is made on glich on our team and then commited to the repo. Then it is pushed out to our Netlify server which hosts the main website. Our discussions can be viewed here(original is here).

7. Isn't it expensive to host a website

Not really. Netlify hosts these websites easily. We are not sponsored by them if you are wondering. Let's do the math. A website like this gets 100gb of bandwith. Each file on average is 10kb. So we can serve this page 10gb/10kb = 10*1024mb/10kb = 10*1024*1024kb/10kb = 1024*1024 = 2^20 = 1048576 times. And cloudflare prevents malicous people from draining the limit.

Staying safe!

Besides helping others make sure you stay safe yourself also. Make sure you do the following

  • Wash your hands even if you are lazy. Don't give the virus a chance to live.
  • Clean surfaces regularly. Viruses like the Coronavirus like to camp on various surfaces *insert fortnite camping picture here*
  • Avoid going to public places too much. We're trying to stop the virus and that means minimizing the spread!
About
